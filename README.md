# Iron Fillword

**Iron Fillword** — это образовательная игра-головоломка на осетинском языке, созданная для изучения языка и расширения словарного запаса.

---

##  1. Правила игры
* На игровом поле (сетка 8×8 или 12×12) случайным образом размещены буквы.
* Задача игрока — найти на поле все осетинские слова из списка, расположенного справа.
* Слова могут располагаться только строго **по горизонтали** (слева направо) или **по вертикали** (сверху вниз).
* Чтобы выделить слово, кликните по его **первой** букве, затем по **последней** и нажмите кнопку **"Афæлварын дзырд"** (Проверить слово). Правильно угаданное слово подсвечивается зеленым.

---

##  2. Функциональные возможности
* **Автогенерация поля:** создание сетки букв под выбранную сложность.
* **Два уровня сложности:** легкий (`ÆНЦОН`, 8×8) и сложный (`УÆЗЗАУ`, 12×12).
* **Интерактивная проверка:** распознавание выделенного слова и подсветка.
* **Игровые метрики:** встроенный таймер и счетчик найденных слов.
* **Сохранение прогресса:** автоматическая запись результатов в текстовый файл.
* **Встроенный словарь:** просмотр полного списка осетинских слов с переводом прямо в интерфейсе.

---

##  3. Руководство администратора
### Системные требования:
* **ОС:** Windows 7 / 8 / 10 / 11.
* **Платформа:** .NET Framework версии **4.7.2** или выше.

### Подготовка файлов:
Для корректной работы приложения в папке с исполняемым файлом `IronFillword.exe` должна находиться директория `Resources/` со следующими файлами:
* `OsetWords.txt` — список слов для генерации (по одному на строку).
* `OsetWordsTranslate.txt` — словарь с переводами в формате `СЛОВО - ПЕРЕВОД`.
* `Results.txt` — файл истории игр (если его нет, программа создаст его автоматически).


---

##  4. Руководство пользователя
1. Запустите файл `IronFillword.exe`.
2. В главном меню выберите уровень сложности и нажмите **"Райдайын"** (Начать).
3. Посмотрите на список слов в правой части экрана и найдите слово на поле букв.
4. Кликните левой кнопкой мыши по **первой букве** слова (она станет серой), а затем по **последней** (весь промежуток выделится).
5. Нажмите кнопку **"Афæлварын дзырд"**.
6. Повторяйте шаги, пока все слова не будут найдены. После этого игра зафиксирует время и сохранит результат.

##  5. Руководство программиста (Структура классов)

Архитектура проекта разделена на графический интерфейс (UI) и бизнес-логику (Core):

### Окна Windows Form
* `FormMenu` — стартовое окно (выбор сложности, переход к словарю и рекордам).
* `FormGame` — основное игровое окно (логика кликов по сетке, таймер, интерфейс).
* `FormDict` — окно просмотра доступных в системе слов и их переводов.
* `FormResult` — форма отображения таблицы рекордов.

### Классы
* `WordPlacer` — отвечает за алгоритм размещения слов на двумерную сетку по горизонтали/вертикали.
* `WordChecker` — распознает клики мыши на слова и проверяет правильность выбранного слова.
* `DictionaryManager` — считывает файл со словами и связывает слова с переводом.
* `WordGenerator` — выбирает случайный набор слов из общей базы для текущего раунда.
* `ResultManager` — отвечает за чтение и запись в файл `Results.txt`.
* `GameUIHelper` — вспомогательный класс для отрисовки интерфейса.
