# IronFillwordProject
Техническое задание к игре “Филворд на осетинском языке”
1. Правила игры
Найти на клеточном поле слова из списка, соединяя буквы по горизонтали/вертикали, слова не пересекаются. После нахождения все слова должны быть вычеркнуты
Основные правила
Поиск слов: Найдите все слова из списка на клеточном поле, заполненном буквами.
Направление: Слова могут идти горизонтально (вправо) или вертикально (вниз).
Не пересекаются: Слова не могут иметь общих букв или пересекаться друг с другом.
Победа: В конце игры должны быть угаданы все слова

2. Назначение и цели разработки
2.1 Назначение программы
Программа предназначена для изучения осетинского языка в игровой форме путём поиска слов в буквенном поле (филворд).
2.2 Целевая аудитория
Игра подходит для детей и взрослых. Для тех, кто владеет осетинским языком и не владеет 
2.3 Цели разработки
Повышение интереса к изучению осетинского языка
Расширение словарного запаса
Развитие логического мышления и внимания
3. Функциональные требования
3.1 Основной функционал
Программа должна обеспечивать:
Отображение игрового поля с помощью DataGridView
Отображение списка слов для поиска с переводом на русский язык
Возможность выбора букв с помощью мыши
Проверку правильности найденного слова
Выделение найденных слов зелёным цветом на игровом поле 
Подсчёт количества найденных слов
Завершение игры после нахождения всех слов
3.2 Игровой процесс
Пользователь запускает приложение
Выбирает уровень сложности (размер игрового поля)
На экране отображается:
поле с буквами
список слов на осетинском языке
таймер
счётчик отгаданных слов
Пользователь выделяет слово щелчком мыши последовательно по каждой букве или щёлкая сначала на первую, а затем на последнюю букву слова
Если слово найдено буквы окрашиваются зелёным цветом на игровом поле 
Игра завершается после нахождения всех слов

3.3 Дополнительный функционал
Таймер
Перезапуск уровня
Словарь
Сохранение результатов
